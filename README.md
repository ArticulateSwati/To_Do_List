# 📝 Smart Todo List 

A sleek, responsive Todo List application built with Vanilla JavaScript. This app focuses on a clean user experience and data persistence using browser LocalStorage.

## 🚀 Live Demo

![To-Do List Demo](./To_do_list.gif)

---

## ✨ Features

* **Data Persistence:** Tasks are saved in `localStorage`, so they stay there even if you refresh or close the browser.
* **Quick Edit:** Double-click on any task text to rename it instantly.
* **Task Management:** Easily add, mark as completed (strike-through), or delete tasks.
* **Keyboard Support:** Add tasks quickly by pressing the `Enter` key.
* **Responsive Design:** Optimized for both desktop and mobile viewing with a modern UI.

## 🛠️ Tech Stack

* **HTML5:** Structured for accessibility and clarity.
* **CSS3:** Custom variables, Flexbox for layout, and smooth transitions.
* **JavaScript (ES6):** Logic for data handling, event listeners, and DOM manipulation.

---

## ⚙️ How It Works

This project demonstrates core JavaScript concepts:

1.  **State Management:** An array of objects `todos` acts as the single source of truth.
2.  **LocalStorage:** The app uses `JSON.stringify()` and `JSON.parse()` to sync the state with the browser's storage.
3.  **Dynamic Rendering:** The `render()` function clears and rebuilds the DOM list whenever the data changes, ensuring the UI is always in sync.
